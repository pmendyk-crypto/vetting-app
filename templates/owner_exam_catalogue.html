National Reference List

Compact reference view for Modality, Study Description, and Exam Code.

List is based on NICIPs and SNOMED CTs used in the DID from 2018-19 to April 2025 combined with the TRUD NICIP Code Set Release (April 2025) and legacy codes from previous versions.

This list includes inactive codes defined by the NICIP Code Set release, and SNOMED CT codes with no clinical mapping from NICIP available.
